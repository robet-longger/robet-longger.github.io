    # 智慧园区解决方案中的智慧安防模块

    # 导读

    # 互联网浪潮下中国已搭上智慧列车急速飞驰，社区作为生活基本单元，经过几十年发展已经步入一个全新时代。智慧社区成为新时代社区发展的新方向。而在智慧社区建设中，安防作为社区安全的第一道屏障，是全社区安全守护者，十分重要。当前智慧社区建设在各大城市展开，安防建设也正在从“传统”向“技防”转型，为社区百姓提供安全与便利，从而推动和谐社区建设
    # 智慧城市的建设已经成为了当今世界城市发展不可逆转的历史潮流，我国智慧城市建设如火如荼，并呈现出了遍地开花的总体建设格局。同时，智慧园区作为建设智慧城市的一个缩影，基于物联网、人工智能的技术下的新智慧园区也在不断的发展着，智慧安防作为智慧园区解决方案中不可或缺的一部分，它的功能也在不断的完善
    # 智慧园区解决方案中的智慧安防的基础应该具备以下几点：

    1. 多系统融合：视频、防盗报警、门禁等业务的有机结合；统一设备适配接口，方便新设备快速接入；平台负责设备事件接收，不做复杂配置，减少平台复杂性。
    2. 智能分析：智能视频、数据分析，获取真正有用的信息;设备与设备、设备与人、设备与业务之间的有机联动;预案的可视化设计和自动执行。
    3. 情景知晓：电子地图、支持GIS、AutoCAD、矢量图片等;视频复核、录像调用、即使回放;关联场景快速切换、报警、事件有效反馈。
    4. 开放拓展：指挥中心快速定位警情，统一调度，统一协调，通过警务联动，消防联动做到警情快速响应;通过信息发布系统实时警情发布给相关人员。

    # 视频监控系统

    1. 视频抓拍：当有报警发生时，系统自动抓拍相应的报警图片，跟报警记录深度绑定。
    2. 录像查询下载：可以根据时间段，特征进行视频深度查询、回放、录像下载。
    3. 视频丢失报警：当有视频丢失时立刻报警。
    4. 人流统计：重要出入口加人流摄像机;实时统计园区内人流量;人流超限报警
    5. 实时视频查看：在手机APP可以实时各看各区域视频。

    # 电子巡更系统

    1. 智慧管控平台对电子巡更的集成主要是记录巡查到位时间、巡查编号、巡查结果、时差等，并可进行特定时间内记录的查询，可以查询巡更路线，值班计划
    2. 自定义：可以在系统自定义多条巡更线路，每个线路上的巡更点生成二维码，打印贴在巡更线路点上，安保人员经过后，通过手机APP进行扫描，即可实现巡更打卡。
    3. 查询功能：通过图形化的界面，可以根据路线、巡更人、时间、未按时巡更等条件查询到巡更信息。
    4. 报表功能：对所有查询到的数据都可以生成报表进行保存或打印。

    # 门禁管理系统

    1. 人脸识别：园区内工作人员，通过人脸登记后，从通道闸或门禁刷脸通行
    2. 访客通行：访客通过移动端等拍照申请，填写相应访问信息，审批通过后，刷脸通行相应区域
    3. 权限管理：根据不同用户，授予不同权限，当有领导进园区参观时，安保人员可以按键放行。
    4. 黑名单：安保人员通过移动端等设置访客人员黑名单，黑名单用户刷脸时，系统自动报警。
