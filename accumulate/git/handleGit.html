# 一.Git密钥配置

# Github 安装

下载 git Windows 版

  • 创建SSH Key：创建ssh key时会提示自定名称和push时的密码（不是git登录密码），一般推荐略过，直接三个回车，如果创建成功会出来一个有图案的小框框。

$ ssh-keygen -t rsa -C "your_email@example.com" // 此处email可任意，不一定要gitLab登录邮箱
1
2
  • 3.）查看SSH Keycopy公钥内容到gitLab里，添加进去。
$ cat ~/.ssh/id_rsa.pub
1
  • 4.）测试SSH Key
$ ssh -T "git@xxx.xxx.com"
